## Proven closure methods

### Harness self-tests and attribution meta-proofs

ADRs 0080 and 0081 factor malformed-case, mutation-failure, safe-reference,
and attribution-harness guards into explicit in-memory self-tests. ADR 0083
makes the attribution self-tests themselves load-bearing across all six
attribution suites. Terminal self-test refusal lines remain named residue when
one more injection layer would only reproduce the same assertion recursively.

### Artifact-mutation vocabularies

ADR 0082 proves the pattern for once-loaded static records: mutate a deep copy
through a closed operation vocabulary, require the intended exact refusal, and
ablate the generator before retaining the case. Apply it to each remaining
inventory, map, disposition, or identity record only after pinning the record
shape and mutation/reason matrix.

### Targeted domain cases

Cross-field comparisons, file-to-file equality, Git-derived lineage, and
resolver semantics need cases constructed from their own predicate. Each case
must be shown to trip the intended guard before retention and then enter the
same refusal-attribution census as its suite. A guard genuinely unreachable
without corrupting the safe baseline stays explicitly unproved; it is not
rounded into coverage.

## Order of work

1. Close remaining mechanical harness and once-loaded-record guards where the
   proven patterns apply without changing domain semantics.
2. Prioritize the highest open domain strata—first-slice, constitutional,
   projection, mathematical, cognitive, and physical—by consequence, not only
   raw count.
3. Retain targeted cases for reachable domain guards and name the exact reason
   for any intentionally unproved terminal residue.
4. Re-run the generalized mutation audit on every checker change; update this
   table only from the retained machine record.

## Boundary

Statement reachability is not correctness. A proved guard is exercised; it is
not shown to encode the right rule. The audit's discovery grammar is itself a
known limitation and must be extended before a new refusal construct can enter
the denominator. Nothing here is an accountable review determination, an
admitted member, independent implementation evidence, or Gate A acceptance.
