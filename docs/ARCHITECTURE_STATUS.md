## Candidate architecture verdict

The leading design is a governed dual-loop research system:

1. a deterministic scientific kernel owns canonical state, contract admission, protocol exposure, authority, resource commitments, evidence custody, verification requirements, claim eligibility, recovery legality, and release settlement;
2. replaceable epistemic/model components propose hypotheses, plans, experiments, code, analyses, critiques, and graph deltas through typed, exposure-bounded contracts;
3. separately assigned verification and rule-bound adjudication stand between generated work and an eligible claim; and
4. private, thesis, and public surfaces are rebuildable non-authoritative projections. No model, cache, receipt, seal, provider response, or UI can publish by implication.

This design fits the strongest retained lessons from the three proof missions. It has not demonstrated autonomous science, scientific superiority, production readiness, or operator acceptance.
